# 🎓 Student Management System

## 📌 Description
A desktop-based application developed using Java Swing that allows students to log in and view their personal details, attendance, and academic performance. Admins can manage student records and upload marks.

## 🚀 Features
- Student Login & Authentication
- View Profile and Attendance
- Semester-wise Results
- Admin Panel (Add/Update/Delete Students)
- Secure Password Hashing (SHA-256)

## 🛠️ Technologies Used
- Java (Swing)
- JDBC
- MySQL

## ⚙️ Setup Instructions
1. Install Java (JDK 8+)
2. Install MySQL and create database `student_db`
3. Add MySQL Connector/J to project
4. Run `Main.java`
